Important HTML tags

This page is designed as a quick reference for some of the common tags covered in Learn Enough HTML to be Dangerous. In the process of making it, we'll learn how to make HTML tables via table and related tags.

The tables below down't include all HTML tags, but they do list many of the most important ones.

Block Elements

Tag Name Purpose
h1–h6 headings include a heading (levels 1–6)
p paragraph include a paragraph of text
table table include a table
tr table row include a row of date
th table header make a table header
td table data include a table data cell
header header specifies the header section
div division creates a division - or section - of the page
ol ordered list creates a numbered list
li unordered list creates an unordered list
blockquote blockquote creates a block quote

Inline Elements

Tag Name Purpose Example Result
em emphasized make emphasized text <em>techincal sophistication</em> techincal sophistication
strong strong make strong text <strong>at least a billion people</strong> at least a billion people
a anchor make a hyperlink <a href="https://learnenough.com/">Learn Enough</a> Learn Enough
img image include an image <img src="https://bit.ly/1MZAFuQ" alt="Michael Hartl">
code code equally size text <code>easier to read code</code> easier to read code
span span specifies an inline element to be styled <span>some text to be styled</span> some text to be styled
li list item creates an item inside an ordered or unordered list <ol><li>I am part of an ordered list</li></ol>

br line breaker creates a single line break some text<br>split up some text

Document Tags

Tag Name Purpose
html HTML Defines the root of an HTML document
head Head Defines information about the document
body Body Defines the document's body
title Title Defines a title for the document
meta Meta Defines metadata about an HTML document
